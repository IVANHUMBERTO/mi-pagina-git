¿Qué son Git y GitHub?

Una guía sencilla para quienes no son programadores.

Primero, hablemos de Git: El "Guardado Mágico"

Imagina que estás escribiendo un documento muy importante, como una novela o una tesis. Cada vez que haces un cambio, guardas una copia con un nombre diferente: "tesis_v1", "tesis_v2", "tesis_final", "tesis_AHORA_SI_FINAL". ¡Es un lío!

Git es como un sistema de "guardado mágico" que soluciona este problema. En lugar de guardar copias enteras, Git guarda solo los cambios que haces. Te permite:

  • Crear "puntos de guardado" (commits): Puedes guardar una versión de tu proyecto en cualquier momento con una descripción de lo que cambiaste.
  • Viajar en el tiempo: ¿Borraste algo por error? ¿La versión anterior era mejor? Con Git, puedes volver a cualquier "punto de guardado" anterior fácilmente.
  • Trabajar sin miedo: Puedes experimentar con nuevas ideas en una "rama" separada sin afectar tu trabajo principal. Si la idea funciona, la unes a tu proyecto. Si no, simplemente la descartas.

Analogía: Git es como la función "Control de Cambios" de Microsoft Word o el "Historial de versiones" de Google Docs, pero mucho más potente y funciona para cualquier tipo de archivo en tu computadora (texto, imágenes, código, etc.).

Entonces, ¿qué es GitHub? El "Club Social" para tus proyectos

Si Git es la herramienta que instalas en tu computadora para llevar el control de tus cambios, GitHub es el sitio web donde guardas y compartes esos proyectos.

GitHub utiliza Git por debajo, pero le añade una capa social y de colaboración:

  • Una copia de seguridad en la nube: Sincroniza tu proyecto de Git con GitHub para tener una copia segura en internet. Si tu computadora se daña, tu trabajo está a salvo.
  • Colaboración sencilla: Invita a otras personas a tu proyecto. Pueden ver tu trabajo, sugerir cambios y contribuir con sus propias ideas de forma ordenada.
  • Un portafolio de tu trabajo: Mucha gente usa GitHub para mostrar los proyectos en los que ha trabajado. Es como un currículum interactivo.

Analogía: Si Git es el programa para escribir tu novela en tu computadora, GitHub es como Google Docs o Dropbox: un lugar en la nube para guardar tu novela, compartirla con tu editor y recibir sus comentarios y correcciones.

En Resumen

Git es el programa que usas en tu computadora para controlar las versiones de tus archivos.

GitHub es la plataforma web donde alojas tus proyectos de Git para respaldarlos y colaborar con otros.

No necesitas ser programador para usarlos. Artistas, diseñadores, escritores y científicos también usan Git y GitHub para gestionar sus proyectos y colaborar de manera más eficiente.
